# Chummer6 Guide Machine Room

This folder holds Chummer6 guide generation inputs that do not belong in the public-facing guide repo.

- `VISUAL_PROMPTS.md`: the visual constitution, scene-family rules, and style-epoch guidance
- `VISUAL_OVERRIDES.json`: per-asset visual overrides for the EA media worker

The public `Chummer6` repo should contain the guide itself and the rendered assets, not the full prompt board that drives them.
