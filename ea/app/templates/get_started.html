Step 1

Choose how this workspace should behave.

The mode changes how the assistant should prioritize work. Pick the one that matches the real operating loop, not the org chart.

Personal

One person, one daily brief, one set of channels. Best when the assistant is helping one operator move faster through inbox, calendar, and follow-ups.

Team / tenant

A shared workspace for a team, support function, or operating unit. Use this when several people need the same activity, drafts, and approvals.

Founder / chief of staff ops

For executive operations with tighter review needs: stakeholder replies, meeting prep, escalation follow-ups, and a more approval-heavy daily loop.

Review posture
