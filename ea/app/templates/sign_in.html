What to expect

Identity first, Google second.

Workspace identity

If your deployment uses company SSO or an access gateway, authenticate there first and the workspace should open with the right identity.

Google connection

Google OAuth is for binding Gmail, Calendar, and contacts to the workspace after access is established. It is not the primary app login.

API tokens

API tokens protect programmatic access and operator automations. They are not a browser registration or password flow.

Private deployments

Private hosts can still enforce host-level access rules. Those checks should feel invisible when the deployment is configured correctly.
